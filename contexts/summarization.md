# Summarization Strategy

## Goal
Keep high-value context with minimal tokens.

## Keep
- durable decisions
- active constraints
- current goals
- open blockers

## Drop
- raw logs
- transient command output
- duplicated instruction text
- stale plans

## Compression
- use concise atomic points
- use bullets when they improve scanability
- use short prose when it is clearer than fragmentation

## Size Targets
- task summary: 3-7 bullets
- decision entry: 1-3 bullets
- blocker entry: 1 line
