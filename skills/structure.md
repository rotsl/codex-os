# Structure Skill

## Purpose
Convert messy input into clear, compact, actionable structure.

## Use When
- input is broad, vague, or noisy
- work needs atomic sequencing
- compression is needed without loss of meaning

## Method
1. state the core problem in one line
2. extract constraints and unknowns
3. split into atomic steps
4. remove low-signal repetition
5. output a tight ordered outline

## Style
- concise and direct
- use bullets when useful
- do not force bullets when a short paragraph is clearer
